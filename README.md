# POLYFILL OBJECT

Simple polyfill of Object.entries and Object.values

## Getting Started

```javascript

<script src="source/polyfill.js"></script>

or

<script src="polyfill.min.js"></script>

```


### Installing

It's simple start :D

```sh

git clone git@github.com:JoHNNyRiver/poyfillObject.git

```
